# Admin Form Components

This directory contains reusable form components for the admin dashboard.

## Components

- **TextInput**: Single-line text input with validation
- **TextArea**: Multi-line text input with auto-resize
- **ImageUpload**: Image upload with drag-and-drop, preview, and validation
- **ArrayInput**: Dynamic array input with add/remove/reorder functionality
- **ConfirmDialog**: Modal confirmation dialog
- **NotificationToast**: Toast notification component

## Features

- Built-in validation display
- Consistent styling across all forms
- Accessibility support
- TypeScript type safety
