# Admin Components

This directory contains Vue components specific to the admin dashboard interface.

## Structure

- **Layout Components**: AdminDashboard, AdminSidebar, AdminHeader
- **Content Editors**: HeroEditor, AboutEditor, ContactEditor
- **Content Managers**: SkillsManager, ProjectsManager, ExperienceManager
- **Dialogs**: ConfirmDialog, NotificationToast

## Usage

These components are used exclusively in the admin section of the application and require authentication to access.
